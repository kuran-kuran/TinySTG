# TinySTG
縦スクロールシューティングゲーム TinySTG です。

https://github.com/kuran-kuran/TinySTG

## はじめに

TinyArcade、PyBadge、Windowsで動作します。
Waveshare19340 を接続した RaspberryPI Pico ではまだ画面サイズの調整がまだだったり音が出なかったりするためテスト版になっています。
